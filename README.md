# 🗄️ Modelagem de Banco de Dados para Faculdade

## 📖 Sobre o Projeto

Este projeto apresenta a modelagem de um **Banco de Dados Acadêmico** desenvolvida utilizando a ferramenta **brModelo**, com o objetivo de representar e organizar as informações de uma instituição de ensino superior.

O documento contém o modelo conceitual e/ou lógico do sistema, demonstrando a estrutura das entidades, atributos e relacionamentos necessários para o gerenciamento de alunos, professores, cursos, disciplinas e demais processos acadêmicos.

---

## 🎯 Objetivos

* Compreender os conceitos de modelagem de banco de dados.
* Desenvolver diagramas utilizando o brModelo.
* Identificar entidades, atributos e relacionamentos.
* Aplicar regras de normalização.
* Estruturar informações de um ambiente acadêmico.
* Preparar a modelagem para futura implementação em um SGBD.

## 🏫 Contexto do Projeto

A modelagem foi desenvolvida para representar o funcionamento de uma instituição de ensino, contemplando informações relacionadas a:

* Alunos
* Professores
* Cursos
* Disciplinas
* Matrículas
* Turmas
* Histórico Acadêmico

O objetivo é demonstrar como os dados podem ser organizados de forma estruturada e eficiente dentro de um banco de dados relacional.

## 📈 Aprendizados

Durante o desenvolvimento deste projeto foram aplicados conceitos fundamentais de análise e modelagem de dados, permitindo compreender como sistemas acadêmicos podem armazenar e relacionar informações de maneira organizada.

A atividade contribui para o entendimento da etapa de planejamento de bancos de dados antes da implementação física em sistemas gerenciadores como MySQL, PostgreSQL ou SQL Server.
